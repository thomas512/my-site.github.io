# my-site.github.io

Pagina web statica che simula il menù di un ristorante.

Progetto personale, realizzato per i miei nonni — non a scopo commerciale.

## Anteprima

Sezioni incluse: Antipasti, Primi, Secondi, Contorni, Dolci.

## Tecnologie

- HTML5
- CSS3 (incluso direttamente nel file, nessuna dipendenza esterna a parte i font)
- [Google Fonts](https://fonts.google.com/): Playfair Display, Cormorant Garamond, Jost

Nessun framework, nessuna build, nessuna dipendenza da installare.
